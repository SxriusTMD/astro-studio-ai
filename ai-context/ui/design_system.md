# Design System — AeroLex AI

Identidad visual:
- espacial
- cósmica
- moderna
- minimalista
- tecnológica

UI:
- dark mode
- glassmorphism moderado
- gradientes suaves
- estrellas animadas

Experiencia buscada:
- visualmente moderna
- simple de entender
- rápida
- limpia
- enfocada en estudio

Principios:
- el chat es el elemento principal
- evitar sobrecarga visual
- mantener claridad
- priorizar legibilidad
- evitar muros de texto

Fuentes:
- sans-serif modernas
- estilo premium/minimalista
